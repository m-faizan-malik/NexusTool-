# NexusTool: EDC Multitools Engineering

NexusTool is all about improving everyday tools through smart, high-quality design. We started with our main product, The Flick, along with a smaller version called The Scout, and we’re working toward building a full range of precise, practical multitools for modern users.
Everything in this project is designed using FreeCAD. One of our main goals is to show that open-source software can be powerful enough to create professional, high-quality tools that can compete in the real market.
Working on such detailed designs is actually what inspired my GSoC 2026 proposal. While using FreeCAD for complex mechanical assemblies, I realized how important it is to have a strong and reliable backup recovery system to support the workflow of the entire community.
This repository contains the mechanical designs and CAD Files for the NexusTool lineup, including the Flick and the Scout, developed entirely in FreeCAD.
